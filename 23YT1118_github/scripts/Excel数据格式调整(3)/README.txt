# 该项目用于实现对育种的结果文件进行格式化地整理工作； 
# 具体要求：
	1. 对每个excel原始表格，都按照特定的要求进行划分sheet（划分标准详见代码中“sheet_name_to_item_dict”变量定义的内容，该内容可能会根据不同的需求更改，在实际使用时，需要和李博联系确认划分方法）；
	2. 在行中，每个出现“All”的行，统一都在该行下面添加下划线；
	3. 在列中，每个性状列，都用竖边框将其包裹起来，并用不同颜色区分不同的性状；
# 使用说明：
	1. Excel对象中定义输入文件路径， 结果文件夹路径， 输入文件的sheet名称， 性状信息起始的列；
	2. python formatting_output_for_xxx.py(xxx表示实验类型，如P1, P3P4, TC。其实不同的xxx文件本质没有区别，只是它们的需要的标准化格式略有不同， 所以在代码中做了细微的调整)。
